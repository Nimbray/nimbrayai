# NimbrayAI V11 — Power Edition

## Objectif

V11 rend NimbrayAI plus puissant sans dépendre d'une API payante obligatoire.

## Améliorations principales

1. Profils de réponse : Auto, Court, Détaillé, Expert, Créatif, Technique, Sources.
2. Conversation Engine amélioré avec plus d'intentions : santé générale, droit général, finance, voyage, code, business, document, recherche.
3. RAG local léger amélioré : découpage, scoring, fiabilité indicative, déduplication.
4. Cache des sources publiques pour éviter des recherches répétées.
5. Sources publiques supplémentaires et mieux organisées.
6. Routeur Ollama et fallback de modèles.
7. Interface conservée : claire, légère, utilisateur à droite, NimbrayAI à gauche.

## Limites actuelles

- PDF/DOCX nécessitent encore une conversion texte ou une future extraction dédiée.
- Le mode demo est utile pour tester l'UX, mais la puissance vient surtout d'Ollama ou d'un provider externe.
- La sauvegarde serveur nécessite Supabase si tu veux persister pour plusieurs appareils.

## Prochaine étape possible

V12 : authentification, comptes utilisateurs, base serveur complète, vraie extraction PDF/DOCX et espace admin.
