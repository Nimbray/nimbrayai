# NimbrayAI V10 — Discussion & Sources Intelligence

## Objectif

Améliorer la qualité des discussions et la qualité des sources sans rendre l’interface compliquée.

## Ajouts principaux

- Conversation Engine V2.
- Détection d’intention : chat, pratique, rédaction, code, business, recherche, document, apprentissage, création.
- Mode de réponse : Auto, Court, Détaillé, Avec sources.
- Actions rapides sur chaque réponse : Copier, Plus court, Plus détaillé, Avec sources.
- Sources publiques supplémentaires : Crossref, PubMed, StackOverflow, docs officielles.
- Sources affichées proprement dans un bloc repliable.
- Contexte interne masqué par défaut.

## Sources gratuites prises en charge

- Wikipédia
- Wikidata
- OpenLibrary
- arXiv
- Crossref
- PubMed
- StackOverflow via StackExchange API
- MDN Web Docs
- Documentation Python
- Documentation Node.js
- Documentation Next.js
- Sources locales uploadées par l’utilisateur
- Base de connaissances NimbrayAI

## Prochaine étape possible

La V10 pourrait ajouter un vrai RAG local avec embeddings, découpage de documents et recherche vectorielle.
