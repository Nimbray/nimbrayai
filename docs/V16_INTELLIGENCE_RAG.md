# NimbrayAI V16 — Intelligence & RAG Upgrade

## Objectif

Rendre NimbrayAI plus intelligent sans coût obligatoire : meilleur moteur conversationnel, meilleure sélection des sources, meilleur fallback Ollama.

## Points clés

- RAG lexical plus strict ;
- déduplication des sources ;
- contexte interne masqué ;
- réponses plus naturelles ;
- sources visibles seulement si demandées ;
- Ollama recommandé avec `qwen2.5:3b`.

## Installation

```powershell
npm.cmd install --no-audit --no-fund
copy .env.example .env.local
npm.cmd run dev
```

## Gratuit

V16 fonctionne en local avec `AI_PROVIDER=demo` ou `AI_PROVIDER=ollama`. Aucun paiement n’est obligatoire pour tester.
