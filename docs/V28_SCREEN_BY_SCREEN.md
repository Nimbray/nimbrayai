# V28 — Cahier écran par écran

## 1. Écran vide du chat

Objectif : ne pas imposer de phrase lourde.

Éléments :
- logo NimbrayAI ;
- titre : “Que veux-tu savoir ?” ;
- phrase courte : “Écris librement. NimbrayAI s’adapte à toi.” ;
- champ d’écriture en bas.

## 2. Conversation

Objectif : lecture confortable.

Règles :
- NimbrayAI à gauche, sans bulle fermée ;
- utilisateur à droite, bulle gris clair ;
- actions sous les réponses très discrètes ;
- sources repliées si présentes.

## 3. Composer

Objectif : interaction premium.

Éléments :
- bouton + pour ajouter une source ;
- textarea large ;
- bouton envoyer ;
- Enter pour envoyer ;
- Shift+Enter pour retour ligne.

## 4. Sidebar desktop

Objectif : historique et fonctions sans surcharge.

Éléments :
- marque ;
- nouveau chat ;
- discussions ;
- compte/mémoire/sources/cerveau/feedback/admin/cloud ;
- statut provider.

## 5. Mobile

Objectif : vraie sensation app.

Éléments :
- bouton menu ;
- drawer latéral ;
- header compact ;
- composer fixe ;
- réponses lisibles ;
- pas de surcharge.

## 6. Cerveau

Objectif : expliquer le système sans complexifier.

Contenu :
- cerveau interne ;
- cerveau externe ;
- domaines couverts ;
- comment enseigner à NimbrayAI.

## 7. Sources

Objectif : permettre d’ajouter des fichiers sans rendre l’interface technique.

Règles :
- upload clair ;
- notes de lecture ;
- suppression facile ;
- sources utilisées discrètement.

## 8. Admin local

Objectif : garder les infos techniques hors de l’expérience publique.

Contenu :
- provider ;
- modèle ;
- Ollama ;
- nombre de sources ;
- nombre de mémoires ;
- export workspace.
