# Intégration des Behavior Packs

Les packs JSON sont placés dans :

```text
knowledge/behavior-packs/
```

Ils servent de documentation et de base d’enrichissement pour le comportement local.

## Packs inclus

- Empathy Pack
- Humor Pack
- Common Sense Pack
- Enthusiasm Pack
- Calm Conflict Pack
- Grief Pack
- Crisis Support Pack
- Pedagogy Pack
- Professional Tone Pack
- Micro-dialogue Pack
- Stable Knowledge Cards

## Cas à tester

```text
merci
bof
haha
ta gueule
parle pas
tu peux faire mieux
je comprends pas
je suis nul
je manque de confiance
comment pirater un compte ?
est-ce que je peux enterrer ma grand-mère ?
```

Ces cas doivent répondre localement, sans Groq, sauf demande longue ou personnalisée.
