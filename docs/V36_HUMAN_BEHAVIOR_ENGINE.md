# NimbrayAI V36 — Human Behavior Engine

V36 intègre les Behavior Packs directement dans le routeur local de NimbrayAI.

## Objectif

Rendre NimbrayAI plus humain, plus vivant, plus sûr et moins dépendant de Groq pour les échanges simples ou émotionnels.

## Nouveaux comportements locaux

NimbrayAI traite localement :

- micro-dialogues : ok, merci, bof, haha, pas mal ;
- critiques : c’est faux, tu peux faire mieux, c’est nul ;
- conflits : ta gueule, parle pas, stop ;
- émotion : je suis nul, je galère, j’ai honte, je manque de confiance ;
- pédagogie : je comprends pas, plus simple, explique mieux ;
- humour léger ;
- cyber-danger : comment pirater un compte ;
- deuil / obsèques.

## Pourquoi c’est important

- moins d’appels Groq ;
- moins de rate limit ;
- réponses plus rapides ;
- personnalité plus stable ;
- sécurité mieux contrôlée ;
- ton plus humain.

## Routeur V36

1. Safety Router
2. Human Behavior Engine
3. Local Brain
4. Practical Local Router
5. RAG / sources si utile
6. Groq seulement si nécessaire

## Règle de ton

NimbrayAI doit être joyeux, chaleureux et vivant dans les conversations légères.  
Dès qu’il y a détresse, deuil, violence, danger ou sujet sensible, NimbrayAI devient calme, protecteur et précis.
