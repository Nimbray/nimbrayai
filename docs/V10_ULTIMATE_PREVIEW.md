# NimbrayAI V10 — Ultimate Preview

Cette version est la meilleure base actuelle de NimbrayAI avant une mise en ligne publique.

## Points forts

- Interface claire, moderne et fluide.
- Messages utilisateur en bulle à droite.
- Réponses NimbrayAI à gauche sans bulle fermée.
- Conversation Engine V2.
- Actions rapides : copier, plus court, plus détaillé, avec sources.
- Mémoire locale.
- Sources utilisateur locales.
- RAG local léger : découpage des fichiers en passages, scoring et sélection des meilleurs extraits.
- Sources gratuites : Wikipédia, Wikidata, OpenLibrary, arXiv, Crossref, PubMed, StackOverflow, docs officielles.
- Ollama compatible et routeur de modèles avec fallback.
- Mode demo utilisable sans API payante.
- Cloud Ready optionnel via Supabase.

## Limites actuelles

- Les fichiers PDF et DOCX ne sont pas encore extraits directement dans le navigateur sans dépendance additionnelle.
- Pour les PDF/DOCX, convertis-les en TXT/Markdown pour les ajouter à la base locale.
- La mémoire et les sources restent locales sauf si Supabase est configuré.

## Recommandation

Utilise cette V10 comme base principale pour continuer le développement.
