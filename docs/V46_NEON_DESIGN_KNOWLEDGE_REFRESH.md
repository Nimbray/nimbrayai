# NimbrayAI V46 — Neon Design & Knowledge Refresh

## Objectif
Faire une version beaucoup plus forte visuellement et plus cohérente dans le cerveau.

## Design
- interface sombre premium ;
- accents bleu électrique + jaune électrique ;
- ergonomie type assistant moderne ;
- sidebar simplifiée ;
- accueil centré sur le chat ;
- composer bas d’écran plus élégant ;
- meilleure lisibilité des messages ;
- mobile plus propre.

## Cerveau
### 1. Behavior Brain
Empathie, micro-dialogues, humour léger, ton humain, réponses chaleureuses.

### 2. Stable Knowledge Brain
Savoirs stables : sciences, histoire, langues, sport, culture, cuisine, notions générales de droit et santé prudente.

### 3. Action Brain
Plans d’action, coaching, organisation, productivité, décisions.

### 4. Safety Brain
Suicide, automutilation, violence, cyber illégal, refus cadrés, redirection utile.

### 5. Document Brain
RAG local, sources utilisateur, synthèse de documents.

### 6. Memory Brain
Préférences utilisateur, style, objectifs, résumé durable.

## Résultat attendu
- NimbrayAI plus pro ;
- plus proche d’un chat moderne haut de gamme ;
- moins de confusion visuelle ;
- plus de cohérence dans les réponses ;
- plus de chaleur humaine.
