# NimbrayAI V60 — Public Beta 2 Mega Edition

## Objectif

Regrouper les versions V51 à V59 en une grosse version de bloc, sans multiplier les petites versions.

V60 part de V50 et consolide :

- stabilité publique ;
- design clair ;
- sécurité ;
- dialogue humain ;
- local-first ;
- documents ;
- projets ;
- cloud ;
- admin ;
- feedback ;
- performance.

## Ce qui change

### 1. Stabilisation V50
V60 conserve le design clair, l’absence de cartes de démarrage, le chat au centre, la sidebar propre, le composer bas d’écran et les bulles utilisateur discrètes.

### 2. NimbrayAI Compass consolidé
Le Compass choisit le style de réponse : calme, humain, direct, coach, prudent, sûr ou créatif.

### 3. Sécurité renforcée
- crise suicidaire : rester avec la personne ;
- automutilation : aide immédiate et ressources ;
- violence : refus + désescalade ;
- cyber illégal : refus + cybersécurité défensive ;
- insultes : limite courte ;
- vulgarité : limite sobre.

### 4. Local-first
Les cas simples et sensibles passent par les moteurs locaux avant Groq.

### 5. Plateforme prête
V60 renforce les bases : comptes, cloud Supabase, documents, projets, admin, feedback.

## Résultat attendu

V60 doit être la meilleure version à montrer à des testeurs publics avant un bloc V80/V100.
