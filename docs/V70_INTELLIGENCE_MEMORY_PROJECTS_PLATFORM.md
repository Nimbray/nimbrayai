# NimbrayAI V70 — Intelligence, Sources, Memory & Projects Platform

## Objectif
V70 rapproche NimbrayAI d'une vraie plateforme d'intelligence personnelle et professionnelle.

La version consolide la V60 et ajoute une direction plus ambitieuse : meilleure continuité de conversation, logique projet, mémoire mieux structurée, sources plus propres et préparation Super Brain.

## Piliers V70

### 1. Intelligence router
NimbrayAI distingue mieux : petite conversation, émotion, sécurité, savoir stable, document, projet, mémoire, code, business, recherche et Super Brain.

### 2. Mémoire
La mémoire doit être contrôlable : voir, corriger, supprimer, exporter.

### 3. Projets
Les projets regroupent : conversations, fichiers, objectifs, décisions, sources et mémoire projet.

### 4. Sources
Les sources restent masquées par défaut. Elles apparaissent seulement si l'utilisateur les demande ou si le mode documentaire le justifie.

### 5. Super Brain
Le mode Super Brain prépare une réponse plus complète : analyse, options, recommandation, plan, livrables et risques.

## Correction importante V70
Le mode silence devient persistant côté interface : si l'utilisateur demande de ne plus répondre, NimbrayAI reste silencieux jusqu'à une demande explicite de reprise.
