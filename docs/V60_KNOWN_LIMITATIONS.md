# V60 — Known Limitations

- Supabase est préparé mais peut rester désactivé si `ENABLE_SERVER_STORAGE=false`.
- Les comptes réels dépendent de la configuration Supabase.
- Groq Free peut limiter les requêtes ; V60 réduit les appels mais ne supprime pas toutes les limites.
- Les réponses juridiques, médicales et financières restent générales.
- Les sources publiques en temps réel ne sont pas branchées par défaut.
- Le multimodal, la voix et l’analyse d’image sont prévus pour des blocs futurs.
