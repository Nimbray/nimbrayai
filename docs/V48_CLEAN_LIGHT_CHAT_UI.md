# NimbrayAI V48 — Clean Light Chat UI

## Axe principal
Refonte de l'interface principale pour se rapprocher d'un chat très épuré : moins de bulles, plus d'air, base claire, accents bleu électrique et jaune électrique.

## Changements clés
- design principal clair ;
- sidebar lumineuse et plus propre ;
- réponses NimbrayAI sans grosse bulle ;
- bulle utilisateur à droite ;
- suppression des cartes de démarrage ;
- écran vide minimal ;
- composer plus fin et plus premium ;
- meilleure lisibilité mobile.

## Vérification
- `npx tsc --noEmit` : OK
- `npm run build` : compilation OK, sandbox interrompu ensuite par timeout pendant la phase finale Next.js.
