# NimbrayAI V12.2 — Adaptive Conversation

Cette version simplifie l’expérience conversationnelle.

## Changements

- Suppression du sélecteur Auto / Court / Détaillé / Expert / Créatif / Technique / Avec sources.
- NimbrayAI choisit automatiquement la longueur et le niveau de détail.
- Les actions de réponse sont simplifiées : copier et régénérer.
- Les sources ne s’affichent que si l’utilisateur les demande explicitement.
- L’interface reste plus proche d’un assistant moderne : écrire, envoyer, recevoir une réponse adaptée.

## Comportement attendu

- Une salutation reçoit une réponse courte.
- Une demande simple reçoit une réponse directe.
- Une demande complexe reçoit une réponse structurée.
- Une demande “avec sources” affiche les sources pertinentes.
