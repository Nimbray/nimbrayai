# Workspaces & Projects

La V45 prépare les projets NimbrayAI.

Un projet peut contenir :
- conversations ;
- mémoire ;
- sources ;
- documents ;
- objectifs ;
- feedbacks.

Exemples :
- projet NimbrayAI ;
- projet business ;
- projet études ;
- projet documents ;
- projet création de contenu.

En mode local, ces informations restent dans le navigateur/export JSON.
En mode cloud, elles pourront être stockées dans Supabase.
