# NimbrayAI V21 — Public AI + Mobile Premium

Objectif : corriger la version publique pour mobile et simplifier l’arrivée dans le chat.

## Changements

- Interface mobile-first.
- Sidebar transformée en menu latéral ouvrable sur mobile.
- Composer fixé proprement en bas avec meilleure gestion de la hauteur mobile.
- Suppression des phrases d’accueil trop directives.
- Suppression des suggestions imposées à l’arrivée.
- Accueil minimal : logo NimbrayAI, question simple, champ de saisie.
- Les numéros de version ne sont plus visibles dans l’interface publique.
- NimbrayAI décide automatiquement de la longueur et du niveau de réponse.
- Les sources restent discrètes et ne s’affichent que si elles sont réellement demandées.

## Checklist mobile

- Tester sur iPhone/Android.
- Vérifier que le clavier ne masque pas le champ de saisie.
- Vérifier l’ouverture/fermeture du menu.
