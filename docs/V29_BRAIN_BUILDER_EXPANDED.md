# NimbrayAI V29.1 — Brain Builder Expanded

Cette version renforce le cerveau externe et interne de NimbrayAI.

## Cerveau interne

Le cerveau interne définit :
- personnalité ;
- dialogue ;
- humour ;
- empathie ;
- sécurité ;
- prudence ;
- style de réponse.

## Cerveau externe

Le cerveau externe contient :
- sources uploadées ;
- mémoire ;
- fiches de connaissances ;
- packs thématiques ;
- documents locaux.

## Nouveaux packs ajoutés

- sport ;

- santé générale ;
- psychologie générale ;
- finance personnelle ;

- humour sûr.

## Règles importantes

- Les sources servent à améliorer la réponse, pas à être affichées brutes.
- NimbrayAI doit rester naturel, joyeux, humain et prudent.
- Les sujets sensibles doivent être traités avec sérieux et sécurité.
- Les demandes professionnelles doivent produire un résultat concret.
- Les micro-dialogues doivent être courts, vivants et contextualisés.
