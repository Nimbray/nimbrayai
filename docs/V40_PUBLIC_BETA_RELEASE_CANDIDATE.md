# NimbrayAI V40 — Public Beta Release Candidate

V40 est une version de consolidation. Elle réunit les briques importantes des versions précédentes pour obtenir une bêta publique plus stable, plus humaine et plus présentable.

## Objectifs

- Garder l’interface claire et minimaliste.
- Masquer les informations techniques aux visiteurs.
- Répondre localement dès que possible.
- Garder Groq pour les demandes complexes.
- Éviter les erreurs techniques visibles.
- Améliorer la cohérence entre les moteurs locaux.
- Préparer des tests publics plus sérieux.

## Pipeline de réponse consolidé

1. Safety Router
2. Micro-dialogue Router
3. Behavior Engine
4. Emotional Variation Engine
5. Action & Coaching Engine
6. Knowledge / RAG Engine
7. Quality Gate
8. Groq uniquement si nécessaire

## Règle produit

L’utilisateur ne doit pas voir la complexité. Il doit voir NimbrayAI : simple, rapide, humain, utile et sûr.
