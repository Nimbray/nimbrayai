# V34 — Format recommandé des fiches

```json
{
  "title": "Pourquoi le ciel est bleu ?",
  "category": "science",
  "summary": "La couleur bleue du ciel vient de la diffusion de Rayleigh.",
  "content": "Explication pédagogique...",
  "keywords": ["ciel", "bleu", "Rayleigh", "atmosphère"],
  "level": "simple",
  "reliability": "élevée",
  "sensitivity": "normale",
  "tone": "pédagogique"
}
```

## Règles

- Une fiche = un sujet clair.
- Le résumé doit être court.
- Les mots-clés doivent refléter les formulations utilisateur.
- Les fiches sensibles doivent déclencher un ton prudent.
- Les fiches de crise doivent être locales et testées.
