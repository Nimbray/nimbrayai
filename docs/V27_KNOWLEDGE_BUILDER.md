# NimbrayAI V27 — Knowledge Builder

La V27 crée la base du vrai cerveau évolutif de NimbrayAI.

## Deux cerveaux complémentaires

### 1. Cerveau interne
Le cerveau interne définit le comportement :

- personnalité joyeuse et humaine ;
- compassion ;
- humour léger ;
- prudence ;
- sécurité sur les sujets sensibles ;
- réponses directes ;
- micro-dialogues naturels.

### 2. Cerveau externe
Le cerveau externe contient les connaissances :

- fichiers Markdown intégrés dans `/knowledge` ;
- sources ajoutées par l’utilisateur ;
- mémoire locale ;
- documents importés ;
- bases publiques gratuites si activées.

## Comment enseigner une connaissance

### Connaissance courte
Ajoute-la en **Mémoire**.

Exemple :

> NimbrayAI doit toujours garder un ton simple, joyeux et professionnel.

### Connaissance longue
Ajoute un fichier dans **Sources** ou crée une fiche `.md` dans `/knowledge`.

Bon format :

```md
# Sujet

Résumé clair.

## Points clés
- point 1
- point 2

## Erreurs à éviter
- erreur 1

## Ton attendu
Simple, utile et prudent.
```

## Règle importante
NimbrayAI ne doit pas afficher les sources internes brutes. Il doit les utiliser pour améliorer sa réponse finale.
