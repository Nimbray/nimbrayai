# Science — explications simples

Une bonne explication scientifique doit commencer simple, puis ajouter les détails si besoin.

## Exemple : ciel bleu
La lumière du Soleil contient plusieurs couleurs. Les molécules de l'air diffusent davantage le bleu que le rouge : c'est la diffusion de Rayleigh. Le ciel paraît donc bleu.

## Règle
Ne pas noyer l'utilisateur. Donner une image mentale claire.
