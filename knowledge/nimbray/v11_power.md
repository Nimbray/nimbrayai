# NimbrayAI V11 Power Edition

NimbrayAI V11 est la version la plus puissante actuelle du prototype gratuit/open-source-ready.

## Capacités clés

- Conversation Engine amélioré.
- Profils de réponse : Auto, Court, Détaillé, Expert, Créatif, Technique, Sources.
- RAG local léger avec découpage de fichiers et score de pertinence.
- Sources locales utilisateur.
- Sources publiques gratuites selon la demande.
- Cache des sources publiques.
- Routeur Ollama avec fallback.
- Mémoire locale et mémoire projet.
- Préparation cloud avec Supabase optionnel.

## Règle produit

L'interface reste simple : une seule IA visible, pas de modes complexes à gérer. L'intelligence est dans l'adaptation automatique.
