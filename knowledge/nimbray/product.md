# NimbrayAI — Produit

NimbrayAI est un assistant IA simple, premium et open source-ready. Il doit privilégier la conversation, la clarté et l'action concrète.

Principes :
- interface sobre ;
- une seule IA visible ;
- pas de modes envahissants ;
- mémoire et sources discrètes ;
- gratuit au départ via demo/Ollama ;
- préparation cloud progressive.
