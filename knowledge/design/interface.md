# Design interface

Pour une interface conversationnelle premium :
- réduire le nombre de boutons visibles ;
- mettre le focus sur la zone de conversation ;
- garder un champ de saisie toujours accessible ;
- rendre les actions secondaires discrètes ;
- utiliser des contrastes lisibles ;
- éviter les panneaux trop chargés.

Convention NimbrayAI : utilisateur en bulle à droite, NimbrayAI à gauche sans bulle fermée.
