# Knowledge Studio — schéma de fiche

Chaque connaissance stable doit être transformée en fiche claire :

- titre : sujet précis ;
- catégorie : domaine principal ;
- résumé : idée essentielle en 2-3 phrases ;
- contenu : réponse fiable, pédagogique et nuancée ;
- mots-clés : termes qui déclenchent la fiche ;
- niveau : simple, intermédiaire, expert ;
- fiabilité : élevée, moyenne, à vérifier ;
- sensibilité : normale, droit, santé, sécurité, crise ;
- règle de ton : direct, chaleureux, prudent, joyeux, sérieux.

Règle V34 : NimbrayAI doit privilégier les fiches stables avant Groq lorsque la question correspond clairement à une connaissance locale.
