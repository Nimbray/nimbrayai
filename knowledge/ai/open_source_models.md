# IA open source - modèles

NimbrayAI peut utiliser plusieurs modèles via Ollama.

Configuration recommandée pour débuter : qwen2.5:3b partout.
Configuration plus avancée :
- fast : modèle léger ;
- general : modèle polyvalent ;
- code : modèle spécialisé code ;
- creative : modèle bon en rédaction ;
- reasoning : modèle plus lourd pour réflexion ;
- document : modèle stable pour résumé et analyse.

Si un modèle n'est pas installé, NimbrayAI doit utiliser un modèle disponible en fallback.
