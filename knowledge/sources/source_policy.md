# Politique de sources NimbrayAI

Les sources doivent aider la réponse, pas encombrer l'utilisateur.

## Priorités

1. Sources utilisateur uploadées.
2. Documentation officielle.
3. Bases structurées : Wikidata, OpenLibrary, PubMed, Crossref.
4. Wikipédia pour contexte général.
5. StackOverflow pour signaux techniques, avec prudence.

## Affichage

- Ne jamais afficher le contexte brut.
- Afficher les sources seulement si l'utilisateur le demande ou si le mode Avec sources est actif.
- Mentionner l'incertitude quand les sources sont faibles ou absentes.
