# NimbrayAI V31 — Smart Router & Local Brain

NimbrayAI doit utiliser son cerveau local avant Groq pour les messages courts, les questions stables et les sujets sensibles.

Ordre de traitement :
1. sécurité humaine ;
2. micro-dialogue ;
3. connaissances stables locales ;
4. sources/RAG si demandé ;
5. Groq seulement si la demande exige de la génération, du raisonnement ou une personnalisation.

Objectif : rapidité, stabilité, économie de tokens et expérience humaine.
