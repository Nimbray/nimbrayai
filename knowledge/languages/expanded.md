# Langues du monde

- familles linguistiques : indo-européenne, afro-asiatique, sino-tibétaine, nigéro-congolaise, austronésienne, ouralienne, etc. ;
- différences entre langue, dialecte, accent, registre ;
- conseils d'apprentissage ;
- traduction simple avec prudence ;
- culture et politesse linguistique.

Bonne pratique : demander le contexte et le niveau si l'utilisateur apprend une langue.
