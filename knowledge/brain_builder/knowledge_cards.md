# Fiches de connaissances NimbrayAI

Une fiche de connaissance doit contenir :
- titre ;
- catégorie ;
- résumé ;
- contenu utile ;
- mots-clés ;
- niveau ;
- fiabilité ;
- limites.

Objectif : enseigner progressivement à NimbrayAI sans réentraîner le modèle.

Règle : une fiche sert à améliorer la réponse, pas à être copiée mot pour mot.
